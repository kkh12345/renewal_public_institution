####   [ 주요 기능 ]
 * 스크롤 시 헤더 고정됨
 * 검색아이콘 클릭 시 검색 모달(pc)
 * 네비게이션 메뉴 호버 시 서브메뉴 드롭다운(pc)
 * 서브메뉴 중 공지사항 클릭 시 공지사항 페이지로 이동
 * 햄버거메뉴 클릭 시 사이드 네비게이션 등장(모바일,태블릿)
